# Accident Severity Prediction (US Accidents)

Multiclass (Severity 1–4) and binary (LOW vs HIGH) prediction using the US Accidents dataset.

## Data

This project uses the Kaggle dataset `sobhanmoosavi/us-accidents` (file: `US_Accidents_March23.csv`).
The raw dataset is **not committed** to this repository.

## Quick start

1. Create a Python environment (3.10+ recommended).
2. Install dependencies (to be added in later commits).
3. Run the dataset download/loader script (added in later commits).
